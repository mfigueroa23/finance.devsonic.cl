# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

To report a security vulnerability, please **do not open a public GitHub issue**.

Instead, use one of the following:
- GitHub private vulnerability reporting (preferred): Security → Report a vulnerability
- Email: support@devsonic.cl

**What to expect:**
- Acknowledgement within 48 hours
- Status updates every 7 days
- A patch release and credit in the changelog once resolved

**Scope:** vulnerabilities in the source code of this repository. Third-party dependency issues should be reported to their respective maintainers.
